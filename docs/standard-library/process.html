Process Module

Import using: use process

Process Identification

  • process.pid() - Current process ID.
  • process.ppid() - Parent process ID.
use process
print("PID: " + process.pid())
print("PPID: " + process.ppid())

Process Control

  • process.exit(code) - Terminate script with exit code.
  • process.kill(pid, signal?) - Send signal to process (default SIGTERM=15).
use process
# process.exit(0)  # terminates immediately
# process.kill(12345, 15)  # send SIGTERM

Working Directory

  • process.cwd() - Current working directory path.
  • process.chdir(path) - Change current working directory.
use process
let cwd = process.cwd()
print(cwd)
process.chdir("..")
print(process.cwd())
process.chdir(cwd)

Command Execution

  • process.exec(command) - Execute shell command. Returns Map with "stdout", "stderr", "exit_code".
use process
let result = process.exec("echo Hello")
print("Exit: " + result["exit_code"])
print("Out: " + result["stdout"])

CLI Arguments & Script Location

  • process.args() - Command-line arguments (array of strings).
  • process.scriptDir() - Directory of the currently executing script.
use process
print(process.scriptDir())
print(str(process.args()))

Asynchronous Spawning

  • process.spawn(command) - Spawn background command, returns ProcessHandle.
  • process.read(handle) - Non-blocking read from process handle.
  • process.close(handle) - Close process handle, returns exit code.
use process
let handle = process.spawn("echo async")
print(str(process.read(handle)))
let code = process.close(handle)
print("Exit: " + str(code))
