Net Module

Import using: use net

Classes & Exports

  • net.TcpServer - TCP Server class. Call new() to create instance, then listen(port), accept(), close().
  • net.TcpClient - TCP Client class. Call new(), then connect(host, port), read(), write(data), close().
  • net.UdpServer - UDP Server class for receiving datagrams.
  • net.UdpClient - UDP Client class. Methods: send(data, host, port), receive(), close().
  • net.Socket - Raw Socket wrapper.
  • net.resolveDNS(hostname) - DNS resolution, returns IP address string.
  • net.URL - URL class for parsing and constructing URLs.
  • net.parseURL(urlString) - Parse URL string into a URL object.

TCP Server Example

use net
let server = net.TcpServer.new()
server.listen(9000)
print("Listening on port 9000...")
let client = server.accept()
client.write("Welcome!\n")
client.close()
server.close()

TCP Client Example

use net
let socket = net.TcpClient.new()
socket.connect("127.0.0.1", 9000)
let response = socket.read()
print("Received: " + response)
socket.close()

UDP Communication

use net
# UDP Server
let server = net.UdpServer.new()
server.bind(9001)
# UDP Client
let client = net.UdpClient.new()
client.send("Hello UDP", "127.0.0.1", 9001)
let data = server.receive()
print("Got: " + data["data"])
client.close()
server.close()

DNS & URL

use net
let ip = net.resolveDNS("localhost")
print(ip)  # 127.0.0.1
let url = net.parseURL("https://example.com:8080/path?q=1")
print(url["host"])  # example.com
print(url["port"])  # 8080
print(url["path"])  # /path
