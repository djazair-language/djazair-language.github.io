# Djazair Documentation

Official documentation website for the [Djazair Programming Language](https://github.com/djazair-language/djazair) — a modern, expressive scripting language built in pure C.

## Structure

- `index.html` — Home page
- `getting-started/` — Installation, first program, running programs
- `language-guide/` — Language fundamentals (variables, types, functions, classes, etc.)
- `examples/` — Example programs (hello world, calculator, todo app, file processing)
- `reference/` — Keywords, operators, builtins, methods, syntax reference, embedding API
- `standard-library/` — Standard library documentation (math, http, json, crypto, etc.)
- `assets/` — Stylesheets
- `generate.py` — Static site generator script
