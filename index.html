The Djazair Programming Language

A modern, expressive scripting language inspired by the soul of North Africa. Blends the expressiveness of high-level scripting with C performance.

Get Started Learn the Language

Why Djazair?

Djazair is a lightweight, embeddable, dynamic scripting engine built in pure C. It features automatic memory management (garbage collection), first-class functions, robust OOP capabilities, async/await coroutines, pattern matching, and a comprehensive standard library. It is designed to be easily embedded in C/C++ projects, or run as a standalone interpreter for automation, tooling, game development, or web scripting.

⚡
Dynamic Scoping & Types
Automatic type inference via let declarations, with native collections (Arrays & Maps) ready out-of-the-box.
🚀
Asynchronous Concurrency
First-class async/await coroutines. Run concurrent workflows, networking tasks, and file operations efficiently.
🏛️
Object-Oriented Programming
Fully supports OOP paradigms including classes, single inheritance, constructors, superclasses, and polymorphism.
🔋
Batteries Included
Rich standard library including regular expressions, cryptography (SHA-256), TCP/UDP networking, directories, threads, and math.

Quick Snippet Preview

Here is a taste of Djazair showing functions, dynamic string interpolation, and standard library modules:

use math

fn computeArea(radius)
    return math.PI * math.pow(radius, 2)
end

let r = 5
print("The area of a circle with radius ${r} is ${computeArea(r)}")
# Output => The area of a circle with radius 5 is 78.5398
