Why Djazair?

A lightweight engine that punches above its weight — productive, embeddable, and ready for real-world use.

Dynamic Typing & Inference
Write less boilerplate with let declarations, native Arrays and Maps, first-class functions, and closures — productive from the first line.
Async/Await Coroutines
First-class coroutines for non-blocking I/O, concurrent workflows, and efficient async scripting — no callback spaghetti.
Object-Oriented Programming
Full OOP with class, single inheritance, constructors (init), super, and polymorphism — build organised code at any scale.
Pattern Matching
Expressive match/case/default constructs for clean, declarative branching — more readable than chains of elif.
Embedding-First C API
Pure C with zero external dependencies. Embed into any C/C++ project via a minimal, documented API — ideal for extending applications.
Batteries Included
Comprehensive standard library: regex, TCP/UDP networking, JSON, crypto (SHA-256), threads, file I/O, date/time, math, UUID, and more.
Robust Error Handling
Structured try/catch/finally with throw for reliable error recovery — write production-ready scripts.
Automatic Memory Management
Precise tracing garbage collector — no reference counting, no manual free(). Focus on your logic, not memory.
