How DPM Works

The Djazair Package Manager (DPM) uses a modern, decentralized architecture. Packages are fetched directly from the official extensions repository or any public GitHub repository. When you install a package, DPM downloads it securely into your global Djazair libraries directory, making it instantly available to all your scripts.

Install Packages
Install globally from name, GitHub, or ZIP files.
dpm install <pkg>
Build Native Code
Compile a package's native C/C++ extensions.
dpm build <pkg>
Update Dependencies
Update installed packages from original sources.
dpm update <pkg>
Package Scaffold
Initialize a new package and generate dpm.json.
dpm init
